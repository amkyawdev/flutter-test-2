# AMK Developer Website

A modern personal portfolio built with Flutter Web featuring smooth animations and a sleek dark theme.

![Flutter](https://img.shields.io/badge/Flutter-3.16+-02569B?style=flat-square&logo=flutter)
![Dart](https://img.shields.io/badge/Dart-3.0+-0175C2?style=flat-square&logo=dart)
![License](https://img.shields.io/badge/License-MIT-green)

---

## 🚀 Features

| Feature | Description |
|---------|------------|
| **Typewriter Animation** | Hero section with typewriter effect |
| **Dashboard** | Stats cards with staggered animations |
| **Projects Gallery** | Interactive cards with hover effects |
| **Smooth Transitions** | Page transitions with fade effects |

---

## 🛠️ Tech Stack

- **Framework**: Flutter 3.16+
- **Language**: Dart 3.0+
- **Routing**: GoRouter 14.0
- **Animations**: flutter_animate 4.5

---

## 📦 Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/flutter-test-2.git

# Navigate to project
cd flutter-test-2

# Get dependencies
flutter pub get

# Run development server
flutter run -d chrome
```

## 🗂️ Project Structure

```
lib/
```
